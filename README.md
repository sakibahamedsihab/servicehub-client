# ServiceHub — Client

> Multi-vendor appointment booking platform — Next.js 16 frontend.

## Tech Stack

- **Next.js 16** (App Router, Turbopack)
- **React 19 + TypeScript**
- **Tailwind CSS**
- **Better Auth** (`emailAndPassword`, `admin()`, `jwt()` plugins)

## Getting Started

```bash
npm install
cp .env.example .env.local
npm run dev
```

## Environment Variables

See `.env.example` for required variables.
